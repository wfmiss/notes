使用 vuepress (opens new window) 及主题 vdoing (opens new window) ，自动部署在GitHub Pages (opens new window)

纯新手可看 bilibili 的 vuepress 视频教程：https://www.bilibili.com/video/BV17t41177cr (opens new window)

# 准备

VuePress 需要 Node.js (opens new window)>= 8.6

首先需要下载 Node.js

  1. 克隆到本地并进入目录

    git clone https://github.com/oddfar/docs.git && cd docs
    1

    或 gitee 地址：

    git clone https://gitee.com/oddfar/docs.git && cd docs
    1

  2. 安装本地依赖

    npm install
    1

  3. 本地测试

    npm run dev
    1

    默认访问链接：http://localhost:8080/doc

# 快速开始

使用markdown语法编写md文件，所有笔记md文件放在docs/docs目录下

例如添加test类，并编写hello.md文件

  1. 创建目录

    格式：序号+标题

    例如：30.test

  2. 添加笔记

    例如：01.hello.md

  3. 编写内容

    ---
title: 笔记标题
permalink: /test/hello/
date： 2021-01-01 01:01:01
---

## 标题

hello world
    1
    2
    3
    4
    5
    6
    7
    8
    9

    tittle：标题，不填写则默认文件名中的标题，即hello

    permalink：访问链接，不填写则自动生成

    date：日期，默认文件创建时间

  4. 测试运行

    在项目根目录下

    npm run dev
    1

    修改 vuepress 和主题一些配置

  • config\nav.js

    修改首页导航栏
